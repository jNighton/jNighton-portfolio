## Getting Started

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

2. Clone the repo
   ```sh
   git clone git@github.com:mhengerer/jNighton/jNighton-portfolio
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Build page
   ```sh
   npm run build
   ```
